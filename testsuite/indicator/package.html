A set of Junit.org 4 test suites for indicator formats.

All test cases do assume that indicator formats do not defend against missuse and are not trying to use them in an abusive way.

This opens some possibility that tests themselves are by mistake abusive, so to fool-proof those tests the sztejkat.abstractfmt.obj package runs all test suites on own reference implementation both wrapped and not wrapped in CIndicatorXXXXFormatProtector protectors.
