This package contains a test implementation which is just using a Collection of IObjStructFormat0 ( or subclasses ) to implement a stream back-end

The sole purpose of this package is to provide a format-agnostic test bed for inter-operational and unit tests of the engines contained in sztejkat.abstractfmt package.

The classes provided here are not factored in an apropriate way and due to a straight-forward coding do contain many copy & paste code. Do not not follow their example, this is not how coding should be done in production code.
